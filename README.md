# TopographicCorrection

This repository contains [QGIS](https://www.qgis.org/en/site/) plugin with several topographic correction methods implementations.

### Implemented methods

1. Cosine-T
2. Cosine-C
3. C-correction
4. SCS
5. SCS + C
6. Minnaert
7. Minnaert-SCS
8. PBM
9. VECA
10. Teillet regression
